Una guía de los comandos que yo he utilizado para instalar spotify (y spicetify) en mi Arch Linux



Primero instalamos Spotify desde un AUR:

sudo git clone https://aur.archlinux.org/yay-git.git/ /opt/				
sudo chown syltr1x:syltr1x /opt/yay-git
cd /opt/yay-git
makepkg -si
yay -S spotify

Si el viejo spotify no cumple tus necesidades de "ser cool", puedes ponerle un poco de sazón (SPICEtify):

sudo chmod a+rw /opt/spotify/Apps
sudo chmod a+rw /opt/spotify/Apps/*
sudo chown -R syltr1x:syltr1x /opt/spotify
curl -fsSL https://raw.githubusercontent.com/spicetify/cli/main/install.sh | sh

Con eso es suficiente para que puedas presumir de tu spotify tuneado y dejes de ser un pesado con el 'neofetch'
