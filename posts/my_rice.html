Los archivos de configuración se pueden encontrar en syltr1x/dotfiles. Aquí desarrollo las configuraciones de forma mas personal y adjunto algunas imágenes.


- Configuración de Qtile

Instalación de Qtile

Arch Based
Debian Based
pacman -S qtile
sudo apt install xserver-xorg xinit
sudo apt install libpangocairo-1.0-0
sudo apt install python3-pip python3-xcffib python3-cairocffi
pip install qtile

En caso de usar otra distribución. Puede encontrar la guía de instalación en qtile.org


Aplicar la configuración

curl -s https://raw.githubusercontent.com/syltr1x/dotfiles/qtile/config.py -o ~/.config/qtile/config.py


- Configuración de Hyprland (en Arch)

Instalación de yay, y de Hyprland mediante script

sudo git clone https://aur.archlinux.org/yay-git.git/ /opt/
sudo chown tuuser:tuuser /opt/yay-git
cd /opt/yay-git
sudo makepkg -si
sudo git clone https://github.com/soldoestech/hyprland.git
sudo chown tuuser:tuuser /opt/hyprland
cd /opt/hyprland
chmod +x set-hypr
./set-hypr

Aplicar la configuración de Hyprland

curl -s https://raw.githubusercontent.com/syltr1x/dotfiles/main/hypr/hyrpland.conf

- Configuración de Neovim

Para Aplicar mi configuración es necesario tener instalada la ultima version de Neovim

Puedes buscar el archivo que necesites en: neovim/releases


Ejemplo de instalación en linux

tar -xf nvim-linux64.tar.gz
sudo mv nvim-linux64/bin/nvim /usr/bin/nvim
sudo chmod +x /usr/bin/nvim

Para Aplicar la configuración de Neovim

curl -s https://raw.githubusercontent.com/syltr1x/dotfiles/main/nvim/init.vim -o ~/.config/nvim/init.vim --create-dirs


- Configuración de WayBar

Esta configuración esta asociada con la de Hyprland ya que yo los he usado en conjunto


Para ocupar toda la configuración es necesario copiar la carpeta complete

git clone https://github.com/syltr1x/dotfiles
cd dotfiles
mv waybar ~/.config/

Probablemente requieras modificar la linea 78 del config.jsonc para cambiar la zona horaria que mostrara la WayBar



- Configuración de la zsh

Si quieres instalar zsh, oh-my-zsh y powerlevel10k tal cual como yo, solo copia lo siguiente

Arch Based
Debian Based
pacman -S zsh
sudo apt-get install zsh
sh -c "$(curl -fsSL https://raw.githubusercontent.com/ohmyzsh/ohmyzsh/master/tools/install.sh)"
git clone --depth=1 https://github.com/romkatv/powerlevel10k.git ~/powerlevel10k
echo 'source ~/powerlevel10k/powerlevel10k.zsh-theme' >>~/.zshrc
git clone https://github.com/zsh-users/zsh-syntax-highlighting.git ${ZSH_CUSTOM:-~/.oh-my-zsh/custom}/plugins/zsh-syntax-highlighting
git clone https://github.com/zsh-users/zsh-autosuggestions ${ZSH_CUSTOM:-~/.oh-my-zsh/custom}/plugins/zsh-autosuggestions

Puedes también copiar el archivo de la powerlevel10k

curl -s https://raw.githubusercontent.com/syltr1x/dotfiles/.p10k.zsh | tee ~/.p10k.zsh > /root/.p10k.zsh
echo 'typeset -g POWERLEVEL9K_OS_ICON_FOREGROUND=255' > ~/.p10k.zsh
sudo echo 'typeset -g POWERLEVEL9K_OS_ICON_FOREGROUND=196' > /root/.p10k.zsh

Y la configuración de mi zsh. Para esto vas a necesitar agregar las alias para que no de error, o comentar la linea 116

curl -s https://raw.githubusercontent.com/syltr1x/dotfiles/.zshrc | tee ~/.zshrc > /root/.zshrc
