GameHub es una plataforma que reúne precios de juegos obteniendo datos de: Steam, Epic Games, Eneba, Microsoft Store



Método de Instalación:


Windows:

iwr -useb https://raw.githubusercontent.com/syltr1x/syltr1x.github.io/main/gamehub/installer.ps1 | iex
cd gamehub
py -m pip install requirements.txt
py app.py

Linux/Mac OS:

curl -fsSL https://raw.githubusercontent.com/syltr1x/syltr1x.github.io/main/gamehub/installer.sh | sh
cd gamehub
python3 -m pip install requirements.txt
python3 app.py
