Con SSChat podrás ejecutar un servidor para tramitar mensajes entre clientes

❗ Soporta "Bridged-Connection"(LAN to Global) usando ngrok


Modo de funcionamiento (Own Hosting)

* Iniciar el servidor

* Establecer una contraseña (opcional)

* Ejecutar un cliente y conectarse desde otra terminal


Modo de funcionamiento (Third-Party Hosting)

* Conectarse al servidor usando una IP y puerto

* Ingresar la contraseña (en caso de ser necesario)

* Elegir un nombre de usuario para el chat



Puedes encontrar el código y mas información del proyecto en: github.com/syltr1x/mythings/chat
