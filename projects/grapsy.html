Grapsy es una aplicación construida usando Tauri(rust) para autenticarse y administrar almacenamiento en un servidor remoto.

Actualmente se encuentra en la version 0.6, con un esfuerzo constante para brindar mas caracteristicas y eficiencia



Instrucciónes para instalación (usuarios)

Puedes ver los archivos disponibles de la ultima versión aquí



Instrucciónes para compilar (desarrolladores)


git clone https://github.com/syltr1x/grapsy
cd grapsy
npm install
npx tauri dev

Para mas información sobre como usar Grapsy, puedes leer el manual tanto su versión en español como su versión en inglés.
