MailTm nace de la necesidad de una biblioteca que permita hacer peticiones a api.mail.tm


Desarrollé una solución estilo libreria para que otros la usen en sus programas y una CLI lista para usar



Instrucciónes para instalación


Clonar o descargar el repositorio

git clone https://github.com/syltr1x/mailTm

En caso de usar la CLI de python o la libreria debes instalar los requerimientos

pip install colorama requests

En caso de usar la CLI para bash

chmod +x user_mail.sh
